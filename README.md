# Gloam Mobile Character Sheet

A digital, fully responsive, and offline-capable character sheet for the **Gloam Tabletop Role-Playing Game** (v1.02 Ruleset).

## Features
- **Mobile-First Design**: Optimized for phone screens with tap-friendly pips, collapsible grids, and a sticky quick-navigation bar.
- **Offline Persistence**: Uses browser `localStorage` to automatically save your character's stats, inventory, and notes. No account or internet connection required.
- **Rulebook Compliant**: 
  - Tracks Vocation Talents, Arcane Magick, and lifepath events.
  - Automatically calculates Movement Speed and Inventory Capacity based on your Coins stat (including worn gear tracking).
  - Granular hit location, wound, and armor notch tracking.
- **Light/Dark Mode**: Built-in toggle for comfortable reading in any environment.

## Usage
Simply open `gloam_character_sheet.html` in any modern web browser. The sheet will automatically save any changes you make directly to your device's browser storage. You can safely close the tab and return later without losing your progress.

If you need to start over or create a new character, simply scroll to the bottom and click **Reset Character Sheet**.
